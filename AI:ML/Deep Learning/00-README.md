# Deep Learning — Learning Path

Welcome! This folder is structured as a progressive learning path for deep learning, starting from zero and building up to advanced concepts. Read the files in order:

| # | File | What You'll Learn |
|---|------|-------------------|
| 01 | `01-what-is-deep-learning.md` | Big picture — AI vs ML vs DL, where deep learning fits |
| 02 | `02-neural-network-basics.md` | How a neural network is built — neurons, layers, weights |
| 03 | `03-activation-functions.md` | The nonlinear "gates" that make deep networks powerful |
| 04 | `04-cost-and-loss-functions.md` | How the network measures its mistakes |
| 05 | `05-gradient-descent.md` | How the network learns from its mistakes |
| 06 | `06-optimizers.md` | Smarter ways to do gradient descent (Adam, SGD, etc.) |
| 07 | `07-regularization.md` | Preventing overfitting — dropout, batch norm, augmentation |
| 08 | `08-architectures.md` | CNN, RNN, Transformers, GANs — the major model families |
| 09 | `09-deep-learning-vs-ml.md` | When to use DL vs traditional ML |
| 10 | `10-real-world-applications.md` | Industry use cases and practical decision guides |

## Prerequisites

- Basic Python knowledge
- High school math (algebra, basic calculus concepts helpful but not required)
- Curiosity!

## How to Use

- Read sequentially if you're brand new
- Jump to specific files if you need a refresher on a topic
- Each file is self-contained with examples and intuition
